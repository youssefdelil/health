ملاحظة تقنية:

هذه نسخة أولية ثابتة (Static Prototype) بواجهة عربية حديثة،
تستخدم Tailwind عبر CDN و Leaflet للخريطة.

التدفق:
- التسجيل: يحفظ المستخدمين في localStorage مع حالة الموافقة.
- تسجيل الدخول: يوجّه المستخدم للوحة حسب نوعه ويمنع الوصول إن لم يُقبل.
- لوحة المدير: اسم المستخدم admin وكلمة المرور 123455678.
- الصيدلي: إدارة المرضى والمخزون ونشر "طلب عام".
- المورد: إضافة أدوية ورؤية الطلبات العامة.
- المريض: بحث عن الدواء وعرض أقرب صيدليات على الخريطة.

الخطوات القادمة المخطط لها:
- تحويل المشروع إلى Next.js مع مصادقة حقيقية (Firebase/Supabase).
- إضافة Framer Motion و Three.js/Spline لتجارب أكثر.
- بناء API خلفية و تكامل واتساب رسمي.
